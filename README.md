Module 10 – Data Manipulation and Cleaning with Pandas
This repository contains my work for Module 10 of the Python for Data Analysis course.
Assignment Overview
This assignment focuses on cleaning a messy, real-world-style customer dataset for a fictional retail brand called UrbanStyle. The raw data contains inconsistent formatting, duplicates, missing values, and mixed data types — all of which are identified and corrected in this script.
Files Included

module10-assignment.py – Main Python script with all completed tasks

What I Practiced

Identifying and removing duplicate records from a DataFrame
Handling missing values through imputation and removal
Standardizing inconsistent formats for phone numbers, dates, and currency strings
Normalizing text fields (e.g., fixing capitalization in names and categories)
Converting columns to appropriate data types (dates, floats, integers)
Validating cleaned data and generating a data quality summary report
Exporting a cleaned DataFrame for downstream use
